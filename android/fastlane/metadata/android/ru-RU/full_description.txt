Знакомьтесь: Verto.chat — AI-мессенджер, где каждый говорит на своем языке и остается понятым. Зажмите кнопку, скажите фразу, отпустите — и вы сразу получите расшифровку и перевод. При необходимости можно включить озвучку (TTS) по нажатию. Работает рядом на одном телефоне или удаленно в чатах 1:1 и группах.

Почему Verto.chat
• Естественный формат: зажмите микрофон, говорите, отпустите — сообщение появляется с переводом
• Видно обе версии: перевод показывается сразу, оригинал открывается по нажатию
• Озвучка по запросу: TTS генерируется только когда вы нажимаете ▶, чтобы экономить трафик и батарею
• Три режима: Local (один телефон, по очереди), Direct (1:1), Group (несколько участников)
• Несколько языков в одном чате с понятным языком по умолчанию
• Чистый и доступный интерфейс с крупными зонами нажатия

Как это работает

1. Speech-to-text (ASR) превращает голос в текст
2. AI-перевод адаптирует его для другого языка
3. Опциональный TTS воспроизводит естественный голос по нажатию ▶

Подходит для
• Повседневного общения за границей, путешествий и многоязычных семей
• Магазинов, сервисов и небольших команд, которые общаются с клиентами
• Языковой практики и тренировки произношения

Ключевые функции
• Две большие кнопки микрофона для основной языковой пары
• Статусы сообщения: отправка → распознано → переведено → TTS готов
• Переключение «Показать оригинал» / «Вернуть перевод» для каждого сообщения
• Local, Direct и Group чаты с аватарами и языковыми метками
• Приватное хранение медиа и загрузка по защищенным ссылкам
• Простой баланс кредитов для AI-функций (стартовые кредиты включены)

Быстрый старт

1. Выберите два языка
2. Создайте Local, Direct или Group чат
3. Зажмите кнопку, скажите фразу и отпустите — перевод появится сразу
4. Нажмите ▶, если нужно озвучить сообщение

Разрешения
• Микрофон — для записи голосовых сообщений
• Уведомления (опционально) — чтобы не пропускать новые сообщения

Приватность
Ваше аудио и сгенерированные медиа хранятся приватно и доступны только участникам чата. TTS-аудио создается по запросу и безопасно кэшируется.
